Using symbols and equations in R with the powerful symbol.equation.ui R package

Obinna N. Obianom

2023-03-18

Introduction

symbol.equation.ui is a package that provides a user interface for building and inserting equations or symbols within R

Installation and Library Attachment

The symbol.equation.ui package is available on CRAN and can be installed as shown below

install.packages(symbol.equation.ui)

Attach library

library(symbol.equation.ui)

Use

Go to the “Addins” in RStudio and click on the “Symbols and Equations” to launch!

Launch 2: from console

symbol.equation.ui()

Launch 3: from console

symbol.equation.ui2()
